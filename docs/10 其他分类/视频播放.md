# 哔哩哔哩视频播放器框架

一个精简的哔哩哔哩视频播放组件，点击按钮打开播放，关闭停止播放。

## 使用说明

1. 将下方代码保存为 `.html` 文件
2. 用浏览器打开即可运行
3. 修改 `bvid` 参数替换为你想播放的视频

## 完整代码

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>哔哩哔哩视频播放器</title>
    <style>
        * { margin: 0; padding: 0; box-sizing: border-box; }
        body {
            min-height: 100vh;
            display: flex;
            flex-direction: column;
            align-items: center;
            justify-content: center;
            background: linear-gradient(135deg, #1a1a2e 0%, #16213e 100%);
            font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
        }
        .btn-play {
            padding: 16px 40px;
            font-size: 18px;
            color: #fff;
            background: #00a1d6;
            border: none;
            border-radius: 8px;
            cursor: pointer;
            transition: all 0.3s ease;
            display: flex;
            align-items: center;
            gap: 10px;
        }
        .btn-play:hover {
            background: #008ac1;
            transform: translateY(-2px);
            box-shadow: 0 8px 25px rgba(0, 161, 214, 0.4);
        }
        .btn-play::before {
            content: '▶';
            font-size: 14px;
        }
        .modal {
            display: none;
            position: fixed;
            top: 0;
            left: 0;
            width: 100%;
            height: 100%;
            background: rgba(0, 0, 0, 0.9);
            z-index: 1000;
            align-items: center;
            justify-content: center;
            flex-direction: column;
        }
        .modal.active { display: flex; }
        .video-wrapper {
            position: relative;
            width: 90%;
            max-width: 1000px;
            aspect-ratio: 16/9;
            background: #000;
            border-radius: 12px;
            overflow: hidden;
        }
        .video-wrapper iframe {
            width: 100%;
            height: 100%;
            border: none;
        }
        .btn-close {
            margin-top: 20px;
            padding: 12px 30px;
            font-size: 16px;
            color: #fff;
            background: rgba(255, 255, 255, 0.1);
            border: 1px solid rgba(255, 255, 255, 0.3);
            border-radius: 6px;
            cursor: pointer;
            transition: all 0.3s ease;
        }
        .btn-close:hover {
            background: rgba(255, 255, 255, 0.2);
        }
    </style>
</head>
<body>
    <button class="btn-play" onclick="openVideo()">播放视频</button>
    
    <div class="modal" id="modal">
        <div class="video-wrapper" id="videoContainer"></div>
        <button class="btn-close" onclick="closeVideo()">关闭播放</button>
    </div>

    <script>
        // 配置：修改这里更换视频
        const BVID = 'BV1GJ411x7h7';  // 替换为你的BV号
        const PAGE = 1;  // 分P视频的页码
        
        function openVideo() {
            const container = document.getElementById('videoContainer');
            container.innerHTML = `<iframe 
                src="https://player.bilibili.com/player.html?bvid=${BVID}&page=${PAGE}&autoplay=1" 
                allowfullscreen>
            </iframe>`;
            document.getElementById('modal').classList.add('active');
        }
        
        function closeVideo() {
            document.getElementById('modal').classList.remove('active');
            document.getElementById('videoContainer').innerHTML = '';
        }
        
        // 点击遮罩关闭
        document.getElementById('modal').addEventListener('click', function(e) {
            if (e.target === this) closeVideo();
        });
        
        // ESC键关闭
        document.addEventListener('keydown', function(e) {
            if (e.key === 'Escape') closeVideo();
        });
    </script>
</body>
</html>
```

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `BVID` | 视频的BV号 | `BV1GJ411x7h7` |
| `PAGE` | 分P视频的页码 | `1` |
| `autoplay` | 自动播放 | `1` 开启，`0` 关闭 |

## 获取BV号方法

1. 打开哔哩哔哩视频页面
2. 复制URL中的BV号（如 `https://www.bilibili.com/video/BV1GJ411x7h7`）
3. 将 `BV1GJ411x7h7` 替换到代码中的 `BVID` 变量

## 功能特点

- ✅ 点击按钮打开视频弹窗
- ✅ 关闭按钮/遮罩/ESC键关闭播放
- ✅ 关闭时自动停止播放（移除iframe）
- ✅ 响应式设计，适配移动端
- ✅ 毛玻璃效果，美观简洁
